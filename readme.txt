MathTutor AI - Pro
Profesjonalna strona do zarządzania korepetycjami z matematyki.

Struktura plików:
- index.html: Główny plik wejściowy
- assets/: Kod źródłowy aplikacji (React, TypeScript)
  - components/: Komponenty UI
  - services/: Logika biznesowa i integracje (Gemini AI)
  - types.ts: Definicje typów
  - App.tsx: Główny komponent aplikacji
  - index.tsx: Punkt wejścia React

Instalacja i Uruchomienie:
Aplikacja wykorzystuje esm.sh do ładowania zależności, więc nie wymaga 'npm install' w środowisku przeglądarkowym obsługującym moduły ES.
Wymagane jest ustawienie zmiennej środowiskowej API_KEY dla funkcjonalności AI.
